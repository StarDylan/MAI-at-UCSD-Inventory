Bulk Edit Inventory

Instructions

Use this tool to make bulk edits to your inventory. The process works as follows:

  1. Download a spreadsheet of your entire inventory
  2. Make changes to the spreadsheet (edit details, move stock items between items, etc.)
  3. Upload the modified spreadsheet
  4. Review the changes that will be made
  5. Confirm and execute the changes

Important: All changes will be logged in the audit trail with your username.

Step 1: Download Inventory

Download a spreadsheet containing all your inventory items and stock items.

The spreadsheet includes instructions on how to make changes.

What You Can Do
Edit Item Details
  • Change item names, manufacturers, GTINs, etc.
  • Update tags (must use existing tags)
  • Modify cost information and other properties
Edit Stock Items
  • Update quantities, locations, details, etc.
  • Change expiration dates and other properties
Move Stock Items Between Items
  • Change the Item ID of a stock item to move it
  • Combine multiple items by moving all stock to one item
Remove Items
  • Items with no stock items will be removed
  • Set stock item quantity to 0 to remove it
